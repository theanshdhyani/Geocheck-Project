Legal & Compliance
Protocol Matrix

Under Indian Civil and Criminal Law, geotechnical misrepresentation is treated as corporate fraud. GeoCheck Pro logs serve as court-admissible forensic evidence.

The Consequence Escalation Matrix

Stage 1: Forensic Anomaly & Location Lock

Initial Verification

Triggered when reported parameters deviate significantly from site-specific physical limits. The specific location is permanently locked from re-registration to prevent data manipulation.

Governing Code

NBC 2016 Part 6 / IS:1892

Impact

Report Rejected & Location Locked

Stage 2: Firm Forgery Alert

Intentional Manipulation

Triggered when the same Contracting Firm is caught submitting fraudulent geotechnical reports across multiple distinct project locations.

Governing Code

RERA Act, 2016 (Section 14)

Impact

Immediate Asset Freeze & Affidavit Review

Stage 3: Criminal Prosecution

Blacklist Event

Permanent ban of the firm following three separate incidents of location fraud. Documentation is forwarded for structural endangerment charges.

Governing Code

BNS (Sec 467/468)

Impact

7-10 Year Term & Full Demolition Order
