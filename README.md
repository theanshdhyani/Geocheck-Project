# GeoCheck 🌍

An AI-powered web application designed to detect fraudulent soil investigation reports.

## Overview
GeoCheck provides an interactive UI that analyzes soil investigation data and verifies document authenticity. By utilizing SHA-256 hashing, the system ensures data integrity and flags manipulated or copy-pasted reports that could compromise construction safety.

## Features
- **Fraud Detection:** Identifies inconsistencies in soil reports using AI.
- **Cryptographic Verification:** Uses SHA-256 hashing to secure and verify report data.
- **Interactive Interface:** Clean, browser-based UI for quick report scanning.

## Tech Stack
- **Frontend:** HTML, CSS, JavaScript
- **Deployment:** Netlify
